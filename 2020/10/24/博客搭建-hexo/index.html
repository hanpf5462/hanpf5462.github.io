hexo搭建个人博客

前言

  • 平时自己记录笔记的习惯不是很好，一般都喜欢将笔记记录在本地，比如office的OneNode，还有一个本地数据库软件MyBase，效果一直不是很理想。
  • 作为一个的嵌入式后端码农，希望通过此次博客搭建，以后所有的学习文档，都在此分享，一方面督促自己学习，一方面跟大家互相讨论，如上！

环境

  • Centos-8系统
  • 目标使用gitpage搭建自己的博客网站，可以随时随地访问。

部署本地环境

安装node.js

sudo yum -y install nodejs
## 安装好后查看版本号,node 、npm
node -v
v10.21.0
npm -v
6.14.4

安装cnpm

  • 跟npm一样，不过是国内源，速度快很多
npm install -g cnpm --registry=https://registry.nmp.taobao.org

安装hexo框架

cnpm install -g hexo-cli
## 安装完成后
hexo -v

myblog创建

mkdir myblog
cd myblog
# 初始化基础hexo信息
hexo init

编写第一篇博客

hexo n "my_first_blog"
########## 
INFO  Validating config
INFO  Created: /home/hanpengfei/harddisk/myblog/source/_posts/my-first-blog.md
## 通过修改.md文档更改博客文章内容

配置github,本地启动和gitpage启动

  • 新建repository
    • 登陆github，新建respository

安装git支持

cd myblog
cnpm install --save hexo-deployer-git

配置git

vim _config.yml
##在最后deply添加
deploy:
   type: git
   repo: https://github.com/hanpf5462/hanpf5462.github.io.git
   branch: master

启动服务

hexo s
##开启本地服务 http://xxx.xx.xx.xxx:4000，调试用，调试完成推送到github上即可上即可
#推送到远程
hexo d
#通过github访问
https://hanpf5462.github.io

图片插入

本地图片插入方法

  • _config.yml文件修改

    #该选项要修改位true
post_asset_folder:true

  • 首先需要安装支持解析图片的插件

    ## 注意不要使用cnpm install hexo-asset-image --save
## 这个版本内容不对 会导致失败 切记！！！
cnpm install https://github.com/CodeFalling/hexo-asset-image --save

  • 在本地创建相应文件夹存放图片，一般与md文件名一致

    ├── hexo搭建个人博客
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── hexo搭建个人博客.md

  • md中书写格式（相对路径）
